### 操作方法

- 1つ目の追加した機能はRボタンを押すことで始めの位置にリセットするというもの。
2つ目の追加した機能はSを押すことでこうかとんの速度が上がる。
3つ目の追加機能はnを押すことでそのマスのこうかとんを残すことが出来る。
